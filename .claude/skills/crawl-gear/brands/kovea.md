# 코베아 (kovea)

- **공식**: https://www.kovea.co.kr  (Cafe24, JS 렌더 → puppeteer 필요)
- 크롤러: `sites/kovea.js`(리스팅+스펙이미지), OCR `kovea-specs.py`/`kovea-fixweight.py` 등
- 마지막 크롤: 2026-05, 317개

## ⚠️ 주의사항
1. **스펙/무게가 본문 텍스트가 아니라 긴 상세 이미지 안**(koveaimage.cafe24.com). → macOS Vision OCR.
   - 한국 "상품 정보 제공 고시" 표(크기/중량/재질/수용인원/내수압)가 **마지막 이미지 하단**(부속 붙으면 끝에서 2~3번째).
   - **침낭 온도는 이미지 중간**(ISO 표). 세탁온도(물 30°C)는 제외.
   - 긴 이미지는 ~1500px 타일로 잘라 OCR(통짜 금지). 한국어는 fast 레벨만 지원.
2. **무게 파싱 함정**: "중량:970 / 내하중:30kg" → 단위없는 970(=970g) 건너뛰고 30kg(내하중)을 무게로 오인.
   라벨별 sub-값 분리 + 위치(바운딩박스 y) 정렬 필수.
3. **완전수집은 `/sitemap.xml`**(카테고리 리스팅은 품절/단종 누락). 브레드크럼은 카테고리로 못 씀.
4. 색상 3소스 결합: 이름 대괄호 + 이미지 "색상" 항목 OCR + Cafe24 옵션.
