# 시에라디자인 (sierra-designs)

- **한국 공식**: https://sierra-designs.co.kr  (Cafe24, 의류 중심 + 기어)
- **미국 공식**: https://sierradesigns.com  (Shopify, 기어 무게/스펙 소스)
- 크롤러: `sierra-designs.py` (KR+US 하이브리드), `sierra-ocr.py` (KR 기어 무게 OCR)
- companyKorean: 시에라디자인 · 마지막 크롤: 2026-07, 약 681개

## ⚠️ 주의사항 (다시 크롤 시 꼭)
1. **"View All"(카테고리 231)에 기어(텐트/침낭/배낭)가 빠져 있다.** 반드시 기어 전용
   카테고리를 따로 크롤: 텐트 152/153/154/174, 텐트악세서리 155, 침낭 175/198,
   배낭 176·백패킹팩 202·데이팩 367·배낭악세서리 204. (안 하면 스타플라이트 텐트 등 통째 누락)
2. **텐트/침낭은 카테고리 번호로 분류**(이름에 종류어 없음). 배낭류는 이름 키워드로
   (크로스백·사코슈·힙색이 배낭 카테고리에 섞여 있음).
3. **무게는 텍스트에 없고 상세 이미지 안**에 있다 → macOS Vision OCR.
   - **패킹무게 우선**(`패킹 무게` > `총 중량` > `최소 무게`). 최소무게(트레일) 넣지 말 것.
   - OCR 소수점 유실 주의: 예) 미티어 라이트 패킹 1.81kg 를 "181kg"로 읽음 → 30kg 초과는 무시하고
     수동 확인. 하프돔 2=3.2kg / 4=4.2kg 처럼 라벨 없이 "- N.Nkg" 로만 있는 경우도 있음.
4. 파우치/글러브/게이터/체어/소형 데이팩은 **사이트에 무게가 아예 없음**(0). 지어내지 말 것.
5. 겹치는 기어(플렉스 카파시터/미티어/하이루트 등)는 US 사이트 grams 로 무게 보강 가능
   (KR 이름의 괄호 영문 (FLEX CAPACITOR) ↔ US 제목 매칭).
6. 색상은 이름 끝 영문(GREY/WHITE…) → color. `(MULTI)` 처럼 괄호 색상은 미추출됨(보완 여지).
7. Cafe24 사이즈 옵션은 JS 렌더라 561 규모에선 생략(product_no=색상 단위 행).
